# Bairon Sanhueza - Personal Portfolio

Repositorio del código fuente de mi portafolio personal y profesional. 

Como **IT Engineer**, utilizo este espacio para documentar y exhibir mis proyectos principales enfocados en **Machine Learning, Análisis de Datos y Desarrollo Backend/Fullstack**.

## Tecnologías Utilizadas

Este portafolio es una Single Page Application (SPA) construida con:
* **React.js** - Librería principal para la interfaz de usuario.
* **Node.js** - Entorno de ejecución.
* **React Bootstrap** - Framework de diseño responsivo.
* **CSS3** - Estilos personalizados y animaciones.

## Créditos

El diseño visual de esta web está basado en la excelente plantilla open-source creada por Soumyajit Behera, la cual fue adaptada, refactorizada y personalizada para ajustarse a mis necesidades profesionales y stack tecnológico.
